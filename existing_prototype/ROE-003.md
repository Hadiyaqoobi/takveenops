## Context
The skill classifier currently uses pre-trained sentence-transformers for similarity matching.
With 50+ real profiles, we can fine-tune or train a dedicated classifier that understands
refugee-specific skill descriptions (e.g., "I made carpets for 10 years" → textile_manufacturing).

## Technical Notes
- Input: multilingual text from chatbot sessions
- Output: O*NET skill codes + confidence scores
- Consider: zero-shot vs few-shot vs fine-tuned approaches
- Baseline: current sentence-transformer cosine similarity
